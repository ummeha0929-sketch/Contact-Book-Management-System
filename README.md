# Contact Book Management System

A console-based Contact Book built in C++ using Object-Oriented Programming, 
developed as a semester project for the OOP course at University of Sialkot.

## Features
- Add, update, delete, and search contacts (partial keyword match)
- Personal and Business contact types (inheritance + polymorphism)
- Automatic duplicate detection and merging
- Group contacts (many-to-many relationship)
- Search history tracking with timestamps
- Data automatically saved to and loaded from file

## OOP Concepts Used
Encapsulation, Inheritance, Polymorphism (virtual functions), Operator 
Overloading, Function Overloading, Dynamic Memory Allocation, File Handling

## Files
- `cb.h` - class declarations
- `cb.cpp` - function definitions
- `main.cpp` - menu and program flow
